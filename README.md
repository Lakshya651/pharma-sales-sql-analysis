# Pharma Sales SQL Analysis

SQL-based analysis of pharmaceutical sales data using JOINs, aggregations, and subqueries.
This repo includes sample datasets, ready-to-run SQL queries, and example result CSVs for a medium-sized dataset (600 rows).

## Project Structure
```
data/
  medicines.csv
  sales.csv
queries.sql
INSTRUCTIONS.md
results/ (example outputs)
README.md
```

## How to run
Follow instructions in `INSTRUCTIONS.md` to import the CSV files into a SQL client and run `queries.sql`.
